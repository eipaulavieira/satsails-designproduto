# 05 — Variants e Estados

Documentação de todas as variantes e estados de componentes e telas.

## Estados a documentar por componente

| Estado | Descrição |
|---|---|
| Default | Estado padrão de repouso |
| Hover | Cursor sobre o elemento |
| Foco | Elemento selecionado via teclado/acessibilidade |
| Ativo | Pressionado / em ação |
| Erro | Validação falhou |
| Disabled | Indisponível |
| Loading | Aguardando resposta |
| Sucesso | Ação concluída |
| Vazio | Sem conteúdo (empty state) |
