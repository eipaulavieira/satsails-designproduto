# Componentes

Cada subpasta contém o componente com suas variantes e documentação de uso.

## Checklist por componente

- [ ] Variantes documentadas
- [ ] Estados (default, hover, foco, erro, disabled, loading)
- [ ] Versão dark mode
- [ ] Guia de uso (quando usar / quando não usar)
- [ ] Ligação com token de design correspondente
