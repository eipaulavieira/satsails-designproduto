# Card

## Variantes

| Variante | Uso |
|---|---|
| Transação | Item de histórico (envio/recebimento) |
| Ativo | Exibe saldo de um ativo (BTC, DePix, USDT) |
| Informativo | Avisos, dicas, notificações |
| Ação | Card clicável que leva a uma ação |

## Anatomia do card de transação

```
[Ícone tipo] [Título] ............. [Valor]
             [Subtítulo / data]      [Status badge]
```

## Tokens usados

- Background: `--color-surface`
- Border: `1px solid --color-border`
- Border-radius: `12px`
- Padding: `16px`
