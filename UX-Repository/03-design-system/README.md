# 03 — Design System

Sistema de design completo da Satsails: fundação visual, tokens e componentes.

## Subpastas

| Pasta | Conteúdo |
|---|---|
| `fundacao-visual/` | Cores, tipografia, espaçamento, iconografia |
| `tokens/` | Tokens primitivos, semânticos, dark/light mode |
| `componentes/` | Botões, inputs, cards, tabs, modais, badges, toasts, loaders |
| `padroes-de-interacao/` | Estados, animações, feedback de ação |

## Tipografia

- **Arial** — documentação e textos corpo
- **Inter** — interface do app
