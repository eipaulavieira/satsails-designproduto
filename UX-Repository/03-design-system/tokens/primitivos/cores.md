# Tokens Primitivos — Cores

Valores brutos. Não use diretamente nos componentes — use os tokens semânticos.

```css
/* Laranja */
--orange-500: #F7931A;
--orange-400: #FF9800;

/* Vermelho */
--red-600: #CF1C0E;

/* Verde */
--green-500: #00C853;

/* Neutros */
--gray-700: #333333;
--black-900: #0C0B0D;
--white:     #FFFFFF;
--gray-300:  #D1D2D2;
```
