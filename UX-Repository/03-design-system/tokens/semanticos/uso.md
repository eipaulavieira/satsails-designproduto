# Tokens Semânticos

Associam significado de uso aos valores primitivos.

```css
--color-primary:        var(--orange-500);   /* Ações primárias, Bitcoin */
--color-accent:         var(--orange-400);   /* Destaques secundários */
--color-danger:         var(--red-600);      /* Erros, alertas críticos */
--color-success:        var(--green-500);    /* Confirmações, sucesso */
--color-text-primary:   var(--gray-700);     /* Texto principal */
--color-text-inverse:   var(--white);        /* Texto sobre fundo escuro */
--color-bg-dark:        var(--black-900);    /* Fundo app dark */
--color-surface:        var(--white);        /* Superfície cards/modais */
--color-border:         var(--gray-300);     /* Bordas e divisores */
```
