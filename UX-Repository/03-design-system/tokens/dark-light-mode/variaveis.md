# Dark e Light Mode

## Dark Mode (padrão do app)

```css
[data-theme="dark"] {
  --bg:       #0C0B0D;
  --surface:  #1A1A1A;
  --text:     #FFFFFF;
  --border:   #333333;
}
```

## Light Mode

```css
[data-theme="light"] {
  --bg:       #FFFFFF;
  --surface:  #F5F5F5;
  --text:     #0C0B0D;
  --border:   #D1D2D2;
}
```
