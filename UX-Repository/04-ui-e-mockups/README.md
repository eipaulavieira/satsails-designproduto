# 04 — UI e Mockups

Wireframes, mockups em português e inglês, protótipos e microcopy.

## Subpastas

| Pasta | Conteúdo |
|---|---|
| `wireframes/` | Estrutura de telas sem visual final |
| `mockups-pt/` | Versão em português |
| `mockups-en/` | Versão em inglês |
| `prototipos/` | Links para protótipos navegáveis no Figma |
| `microcopy/` | Labels, CTAs, mensagens de erro, onboarding copy |

## Nomenclatura de arquivos

`[secao]-[tela]-[versao].[ext]`  
Exemplo: `onboarding-boas-vindas-v2.png`
