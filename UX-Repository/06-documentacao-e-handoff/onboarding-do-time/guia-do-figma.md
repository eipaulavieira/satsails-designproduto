# Guia do Figma — Satsails

## Organização dos arquivos

Cada arquivo Figma espelha uma pasta deste repositório:

| Arquivo Figma | Pasta GitHub |
|---|---|
| 🎨 Design System | `03-design-system/` |
| 📱 Mockups PT | `04-ui-e-mockups/mockups-pt/` |
| 🌐 Mockups EN | `04-ui-e-mockups/mockups-en/` |
| 🗂 Arquitetura IA | `02-arquitetura-de-informacao/` |

## Nomenclatura de camadas

- Componentes: `[Nome]/[Variante]/[Estado]`  
  Exemplo: `Button/Primary/Default`
- Telas: `[Seção] — [Nome da tela]`  
  Exemplo: `Onboarding — Boas-vindas`
- Tokens de cor: seguem os nomes do arquivo `03-design-system/tokens/`

## Convenções

- Tudo em **português** (exceto termos técnicos do produto)
- Auto Layout obrigatório em todos os componentes
- Variáveis de cor do Figma espelham os tokens deste repositório
