# 02 — Arquitetura de Informação

Estrutura de navegação, nomenclatura e inventário de telas do app.

## Subpastas

| Pasta | Conteúdo |
|---|---|
| `navegacao/` | Sitemap, mapa de fluxos, hierarquia de telas |
| `taxonomia/` | Labels, glossário de termos (DePix, LBTC, PIX com delay...) |
| `inventario-de-conteudo/` | Lista de telas, tipo de conteúdo, status |
| `modelos-mentais/` | Card sorting, tree testing, benchmarks de navegação |

## Seções do app mapeadas

- Onboarding
- Home
- Enviar e Receber
- Compra e Conversão
- Histórico e Analytics
- Suporte e Configurações
