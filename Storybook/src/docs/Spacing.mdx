## Tabela de Referência

| Token | Valor | px | Uso típico |
|---|---|---|---|
| `--space-1` | `4px` | 4 | Gap interno mínimo |
| `--space-2` | `8px` | 8 | Padding de badges e tags |
| `--space-3` | `12px` | 12 | Padding interno de inputs (y) |
| `--space-4` | `16px` | 16 | Padding padrão de componentes |
| `--space-5` | `20px` | 20 | Gap entre elementos de formulário |
| `--space-6` | `24px` | 24 | Padding de cards |
| `--space-8` | `32px` | 32 | Espaçamento de seções menores |
| `--space-10` | `40px` | 40 | Padding de containers |
| `--space-12` | `48px` | 48 | Espaçamento entre seções |
| `--space-16` | `64px` | 64 | Seções maiores de layout |

---

## Regra de Uso

Sempre prefira tokens de espaçamento a valores ad-hoc. A base da escala é **4px** — use apenas múltiplos desta unidade para manter a grade consistente.
